Dealer Network and Customer Experience Analysis

This project aimed to analyze the dealer network and customer experience for a luxury automotive brand, identifying key trends, pain points, and opportunities for improvement.

Key Activities:

  • Collected and cleaned randomly generated data for demonstration purposes from multiple sources, including customer reviews, sales records, and dealer performance metrics.
  • Performed sentiment analysis on customer feedback to categorize reviews as positive, neutral, or negative.
  • Created an interactive Power BI dashboard to visualize key metrics such as sales trends, customer satisfaction scores, and inventory turnover.
  • Analyzed dealer performance to identify underperforming locations and recommend actionable improvements.

Tools and Technologies:

  • Data Analysis: Python (Pandas, TextBlob), SQL.
  • Data Visualization: Power BI.
  • Data Cleaning and Transformation: Excel, Python.
